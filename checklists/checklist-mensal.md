# Checklist Mensal de Performance

Squad: ads-management | Executor: ads-optimizer + ads-strategist
Frequencia: Primeiro dia util do mes | Tempo: 1-2h
Fonte: [SOURCE: Andrew Foxwell — AAA Ascension] + [SOURCE: Molly Pittman — CVJ Review]

## Resultados do Mes
- [ ] Consolidar investimento total por conta/plataforma
- [ ] Consolidar resultados totais (leads, vendas, ROAS)
- [ ] CPA/ROAS medio do mes vs meta acordada com cliente
- [ ] Comparar vs mes anterior: melhorou, estavel ou piorou?
- [ ] Budget executado vs budget planejado (variancia)

## Analise de Funil (por conta)
- [ ] Topo: custo por alcance/impressao dentro do esperado?
- [ ] Meio: engajamento/CTR dentro do esperado?
- [ ] Fundo: taxa de conversao estavel?
- [ ] Retargeting: publico crescendo? Frequencia controlada?

## Criativos — Ciclo de Vida
- [ ] Quantos criativos novos foram testados no mes?
- [ ] Quantos criativos foram aposentados por fadiga?
- [ ] Top 3 criativos do mes (e por que funcionaram)
- [ ] Estoque de criativos para proximo mes: suficiente?

## Estrategia (com ads-strategist)
- [ ] Funil atual esta funcionando? Precisa ajustar?
- [ ] Split de budget entre plataformas ideal?
- [ ] Novos formatos/canais para testar?
- [ ] Metas precisam ser revisadas?

## Relatorios
- [ ] Relatorio mensal gerado para TODOS os clientes (template: relatorio-mensal.yaml)
- [ ] Relatorio enviado/apresentado ao cliente
- [ ] Feedback do cliente documentado

## Planejamento Proximo Mes
- [ ] Metas definidas por conta
- [ ] Testes planejados (criativos, publicos, formatos)
- [ ] Budget confirmado com cliente
- [ ] Calendario de acoes montado

## Veto Conditions
- NAO renovar estrategia sem analisar dados do mes anterior
- NAO enviar relatorio sem comparativo vs periodo anterior
- NAO definir metas sem base em dados historicos
